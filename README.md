# 📄 简历生成器

一个纯前端的在线简历生成工具，填写信息实时预览并导出 PDF。

## ✨ 功能
- 基本信息、教育、工作、项目、竞赛、认证
- 头像上传
- 自定义信息字段
- 技能标签（回车添加）
- 模块拖拽排序、显示/隐藏
- 实时预览 A4 纸效果
- 一键导出 PDF
- 数据保存到本地（刷新不丢失）

## 🚀 本地运行

npm install
npm run dev

## 📦 打包

npm run build
# 生成 dist/index.html，双击即可使用

## 🛠 技术栈
Vue 3 + Element Plus + html2canvas + jsPDF + vuedraggable
